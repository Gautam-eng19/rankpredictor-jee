# rankpredictor-jee
A JavaScript based Fast, mobile-friendly interface web app that predicts JEE Mains rank based on your percentile .

#AS a jee aspirant myself, this was a fun project to start with .
I just wanted to learn how website works. 
I created this with some basic knowledge and slight use of ai

#HOW TO USE-
1)Enter your Jee Mains 2025 Percentile. (eg. 99.14) 
2)This will calculate approximate rank based on your percentile.
